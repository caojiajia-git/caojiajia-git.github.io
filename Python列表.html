列表的格式： 变量的类型为列表，列表中的元素可以是不同类型的。

列表中存放的数据是可以进行修改的，比如“增” “删” “改” “查 ”。

1例如：

list = ['我','爱','你']

print(list)

结果：

​ ['我','爱','你']

2例如：

list = ['我','爱','你']

print(list[1])

结果：

​ 爱

3在for里面用

list = ['我','爱','你']

for i in list:

​ print(i)

结果：

​ 我

​ 爱

​ 你

4在while里面用 加length长度

list = ['我','爱','你']

length = len(list)

i = 0

while i < length:

​ print(list[1])

​ i += 1

结果：

​ 我

​ 爱

​ 你

静态列表：

a = list()

print(a)
