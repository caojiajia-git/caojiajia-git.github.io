python的初级切片

Published: 周三 14 五月 2014
Updated: 周三 14 五月 2014
By 曹佳佳

In Python.

tags: Python列表

列表的格式： 变量的类型为列表，列表中的元素可以是不同类型的。

列表中存放的数据是可以进行修改的，比如“增” “删” “改” “查 ”。

1例如：

list = ['我','爱','你']

print(list)

结果：

​ ['我','爱','你']

2例如：

list = ['我','爱','你']

print(list[1])

结果：

​ 爱

3在for里面用

list = ['我','爱','你']

for i in list:

​ print(i)

结果：

​ 我

​ 爱

​ 你

4在while里面用 加length长度

list = ['我','爱','你']

length = len(list)

i = 0

while i < length:

​ print(list[1])

​ i += 1

结果：

​ 我

​ 爱

​ 你

静态列表：

a = list()

print(a)

    In Python.

    tags: Python切片

    昨天有朋友问了我一个初级的切片问题，让我帮理解一下，
    那今天就简单的为一些初学者解答一下，如有不好还请多见谅哦！

    切片的语法： [起始下标：结束：步长]
    1. 切片是指对操作的的对象截取其中一部分的操作。
    2.字符串,列表，元组都支持切片操作。
    注意：
    选取的区间以“起始”位开始，到“结束”位的前一位结束（不包含结束本身）步长表示选取间隔。 

    a="abcdefg"   
b=a[  :5:2]   /   b=a[ : :-1]  

print(b)   
结果：   
    ace   /   gfedcba

    a="我速度喜看见欢几个你"    
b=a[ : : 3]   
print(b)    
结果：    
    我喜欢你
