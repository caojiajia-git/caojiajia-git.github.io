Django安装命令

Published: 周六 17 五月 2014
Updated: 周六 17 五月 2014
By 佳哥

In Django.

tags: Django命令

安装插件 

pip install Django ==2.1.4

查看版本 

python  -m  Django  -Version

生成django项目（项目名称） 

django-admin startproject 项目名

切换到创建的项目名 

cd 项目名

创建子应用name （子应用名称） 

python manage.py startapp 子应用名

启动项目（在端口输入运行） 

python manage.py runserver

卸载django 

pip uninstall django

数据库迁移： 

python  manage.py  makemigrations        
python  manage.py  migrate

创建超级用户 

python manage.py createsuperuser

在admin.py中注册（把模型类注册到管理员站点） 

先导包
from 子应用名 import models
admin.site.register(models.表名)

  5. python的初级切片

    Published: 周三 14 五月 2014
    Updated: 周三 14 五月 2014
    By 佳哥

    In Python.

    tags: Python切片

    昨天有朋友问了我一个初级的切片问题，让我帮理解一下，
    那今天就简单的为一些初学者解答一下，如有不好还请多见谅哦！

    切片的语法： [起始下标：结束：步长]
    1. 切片是指对操作的的对象截取其中一部分的操作。
    2.字符串,列表，元组都支持切片操作。
    注意：
    选取的区间以“起始”位开始，到“结束”位的前一位结束（不包含结束本身）步长表示选取间隔。 

    a="abcdefg"   
b=a[  :5:2]   /   b=a[ : :-1]  

print(b)   
结果：   
    ace   /   gfedcba

    a="我速度喜看见欢几个你"    
b=a[ : : 3]   
print(b)    
结果：    
    我喜欢你
