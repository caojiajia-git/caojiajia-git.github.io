1、先插入插件static ，在 static 里面包含: js 和 tinymce

2、在添加HTML页面中写入 

 {% block script %}                <!--这个标签script也可以用title-->
        <script src="../static/tinymce/js/tinymce/tinymce.min.js"></script>
        <script src="../static/js/tinymce_setup.js"></script>          <!--富文本用-->
 {% endblock script %}
</head>
<body>
{% block content %}
    <form action="" method="post">
        添加标题：<input type="text" name="title" value=""><br>
        添加内容：<input type="text" name="content" id="rich_content"><br>
        <input type="submit" value="确认添加">
    </form>
{% endblock content %}

  3. python的初级切片

    Published: 周三 14 五月 2014
    Updated: 周三 14 五月 2014
    By 佳哥

    In Python.

    tags: Python切片

    昨天有朋友问了我一个初级的切片问题，让我帮理解一下，
    那今天就简单的为一些初学者解答一下，如有不好还请多见谅哦！

    切片的语法： [起始下标：结束：步长]
    1. 切片是指对操作的的对象截取其中一部分的操作。
    2.字符串,列表，元组都支持切片操作。
    注意：
    选取的区间以“起始”位开始，到“结束”位的前一位结束（不包含结束本身）步长表示选取间隔。 

    a="abcdefg"   
b=a[  :5:2]   /   b=a[ : :-1]  

print(b)   
结果：   
    ace   /   gfedcba

    a="我速度喜看见欢几个你"    
b=a[ : : 3]   
print(b)    
结果：    
    我喜欢你
